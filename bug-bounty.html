BUG BOUNTY PROGRAM

Many security Vulnerability are found all the time. While new vulnerabilities are found it could put others in danger. So the bug bounty program is to help eliminate those flaws. We thank you if you find a bounty by putting in our Hall of Fame and some cash for their achievement.

If you have found a vulnerability make sure to submit it here. For more information you can check what types of vulnerabilities we accept in the rules and FAQ.
